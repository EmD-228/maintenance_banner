# Maintenance Banner Example

This is an example application demonstrating how to use the `maintenance_banner` package.

## Running the Example

1. Make sure you have Flutter installed
2. Navigate to this directory:
   ```bash
   cd example
   ```
3. Install dependencies:
   ```bash
   flutter pub get
   ```
4. Run the app:
   ```bash
   flutter run
   ```

## Features Demonstrated

- Basic usage of `MaintenanceWrapper`
- Toggle maintenance mode on/off
- Customizable banner appearance
- State management with setState

## Usage

The example shows how to:
- Wrap your app with `MaintenanceWrapper`
- Use a controller to manage maintenance state
- Customize the banner's appearance (message, colors, height, icon)
